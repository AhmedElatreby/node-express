node express CRUD
===

## Features
- Setup & Install / Simple Server
- Simple API
- Routes, Request & Response
- Express Router
- Static Files
- Template Engine Setup With Views
- Middleware
- Nodemon


## Quick Start
```bash
# Install dependencies
npm install

# Serve on localhost:5000
npm run dev
```
